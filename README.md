# Anagram Solver

Веб-додаток для гри в анаграми, реалізований на чистому JavaScript (ES Modules).

##  Функціональність

- Генерація анаграм  
- Введення відповіді та перевірка  
- Таймер раунду  
- Підказки  
- Сторінка результатів  
- Cookie banner (збереження згоди користувача)  

##  Запуск проєкту

Встановлення залежностей:

```bash
npm install
